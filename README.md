# BarbeariaLV2 - Sistema de Agendamento para Barbearias

## Sobre o Projeto
O BarbeariaLV2 é um sistema de agendamento online desenvolvido em PHP, criado especialmente para atender às necessidades de uma barbearia específica.  
O objetivo do projeto é otimizar o gerenciamento de horários, oferecendo uma plataforma moderna e acessível tanto para os clientes quanto para os administradores da barbearia.

Através do sistema, os clientes podem realizar agendamentos de forma prática, enquanto os barbeiros e administradores têm acesso a uma interface organizada para visualizar, confirmar e gerenciar os atendimentos.

## Objetivos
- Oferecer uma interface simples e intuitiva para o agendamento de horários;  
- Permitir o cadastro de clientes, barbeiros e serviços;  
- Facilitar o controle de disponibilidade e horários;  
- Gerar relatórios e histórico de agendamentos;  
- Personalizar o sistema conforme a identidade da barbearia parceira.
